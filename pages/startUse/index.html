# 开始使用

# 整体介绍

综合门户2.0系统，是基于综合门户1.0系统版本原有需求的基础上，采用前端最新Vue框架重构，引入组件化开发模式，除了保留原有系统的栏目及布局功能外，新增菜单配置、主题切换、门户授权、统一用户登录、友情链接维护、系统配置功能维护等功能，布局及栏目具有更大自定义空间，可灵活配置各类型及渠道栏目，也可自行开发组件及代码注入等方式进行栏目模块的自由开发。

# 适用对象及场景

有门户网站展示需求的项目或组织机构，便于展示新闻类信息、通知类信息、相关待办统一入口、汇总各日常使用应用或常用链接组织统一入口等。

# 产品特色

整个系统分为前台及后台两种使用场景，前台主要是针对单个系统用户进行内容展示、查看或相关应用的入口展示、待办审批流程展示及日程的制定等，方便各类型及不同权限用户进行不同的操作；后台主要针对管理员进行相关新闻、应用、模块、权限、门户授权、系统配置等操作，可快速配置及定制网站或门户，进行快速部署。
