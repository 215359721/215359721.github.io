# 导航菜单

导航菜单为网站的重要组成部分，是一个网站及门户重要功能的目录，并提供关键功能及路径的快捷入口，本系统可以根据需求需要提供不同门户的菜单定制，即不同门户展示不同的菜单，或不展示菜单等需求，最大化满足门户业务需要

# 菜单初始化

菜单管理 -> 选择对应门户 -> 创建菜单 -> 输入初始化菜单数量（一级菜单）-> 完成创建

# 菜单添加

菜单管理 -> 点击“+”添加一级菜单（或点击一级菜单下“添加子菜单”添加对应的二级菜单）-> 输入对应菜单必填项 -> 点击“确定”完成创建

注意

1.【内链】链接模块，使用路由形式:“/路由地址”

2.【内链】内嵌iframe页面，包含头部及尾部，使用“http://链接地址”

3.【外链】当前页面直接替换为url，不包含头尾，使用“http://链接地址”

# 菜单修改

菜单管理 -> 鼠标指向要修改的菜单 -> 弹出框中点击“修改”按钮 -> 修改对应菜单信息 -> 点击“确定”完成创建

# 菜单顺序调整

菜单管理 -> 鼠标指向要调整顺序的菜单 -> 点击“<”或“>”进行前移或后移完成菜单顺序调整

# 菜单删除

菜单管理 -> 鼠标指向要删除的菜单 -> 弹出框中点击“移除”按钮 -> 点击确认框“确定”完成菜单的删除

注意

新建门户后，特别是多门户并存时，默认每个门户对应的菜单为空，即新建门户后，门户默认没有菜单，如需设置，需要对门户先进行菜单初始化操作

同样若已为门户设置了菜单，但后续不在需要显示菜单，则需将对应门户下菜单全部删除即可
