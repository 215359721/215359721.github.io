# 网站头部尾部

# 头部

头部采取公共组件嵌入的方式，在各个页面进行展示（首页、二级页、三级页），修改头部文件一处可以在所有头部调用处都生效

头部包含三部分：上方信息展示区域、中部功能区域、下方菜单展示区域

注意

上方、中部区域为固定显示区域，不受数据及门户区别控制

下方菜单区域根据后台是否为此门户设置了菜单，若未设置菜单，则菜单区域不显示

# 上方信息展示区域

展示当前时间、当前登录用户、用户消息、换肤功能入口、退出链接

信息展示区域高度固定，消息数据随用户登录信息展示不同条目，消息、换肤功能为鼠标指向该区域自动展示下拉框信息内容

# 中部功能区域

左边展示当前网站logo（目前为固定文件，后续会改为后台可维护图片文件）

右侧展示搜索功能，搜索功能输入关键词自动链接至航天内部搜索引擎，搜索引擎地址目前后台地址可维护，详细操作请参照后台“系统配置”功能

# 下方菜单展示区域

菜单区域展示目前后台已为当前门户配置的菜单，菜单最多支持二级，点击菜单跳转至相应功能

# 换肤功能

换肤功能提供包括默认主题在内的4种主题的切换（默认、航天蓝、优雅黑、中国红）

用户可自行切换喜好的皮肤主题，切换后，下次使用同一浏览器打开会继续使用之前切换的主题皮肤样式

注意

用户切换使用的皮肤主题及相关样式是保存在浏览器本地缓存中

用户切换浏览器、清除浏览器本地数据及缓存后，用户切换的皮肤主题会一同被清除，并恢复默认主题皮肤

# 尾部

尾部同头部一样采取公共组件嵌入的方式，在各个页面进行展示（首页、二级页、三级页），修改尾部文件一处可以在所有尾部调用处都生效

尾部包含三部分：左侧企业信息展示区域、中部友情链接区域、右侧app下载地址展示区域

注意

尾部目前有两种展现形式：1、使用默认尾部 2、后台配置html自定义尾部

以上介绍为默认尾部，后台配置html自定义尾部详见“系统配置”

# 左侧企业信息展示区域

左侧企业信息展示区域显示企业常用联系方式及地址信息（目前为固定信息，后续版本陆续改为后台动态配置）

# 中部友情链接区域

友情链接配置目前为读取后台“友情联系”功能中维护的相关友情链接类别及类别下的友情链接单位，点击可直接跳转至相关单位url地址

如何配置友情链接请参照“友情链接”功能

# 右侧app下载地址展示区域

app下载地址展示安卓及iosapp下载二维码地址，为固定图片，如需更换请提供最新图片

此功能仅做展示，不做动态维护
