# 友情链接

友情链接提供相关单位及指定分类的链接访问、维护，目前使用场景为默认页尾处友情链接，后续可能开发更多使用场景

友情链接分纯文字链接与图片链接两种形式，目前仅使用纯文字链接，建议后台创建链接时选择文字链接，图片链接目前可以创建，但无使用场景

友情链接后台提供友情链接分类的增删改查，友情链接的增删改查功能（某一友情链接隶属于某一分类）

# 分类的增删改查

# 1.查看

友情链接 -> “查看分类”按钮

# 1.新增

友情链接 -> “添加分类”按钮 -> 填写分类信息 -> 点击“确定”完成创建

# 3.修改

友情链接 -> “查看分类”按钮 -> 某一条数据操作“修改” -> 弹框修改分类信息 -> 点击“确定”完成修改

# 4.删除

友情链接 -> “查看分类”按钮 -> 某一条数据操作“删除” -> 弹框确认删除

# 友情链接增删改查

# 1.查看

友情链接 -> 默认列表

# 2.新增

友情链接 -> “添加链接”按钮 -> 填写链接信息 -> 点击“确定”完成创建

注意

图片链接目前暂无使用场景，目前不建议创建图片链接

# 3.修改

友情链接 -> 某一条数据操作“修改” -> 弹框修改友情链接信息 -> 点击“确定”完成修改

# 4.删除

友情链接 -> 某一条数据操作“删除” -> 弹框确认删除

注意

初次使用，特别是尚未创建友情链接时，应先对分类进行管理，先创建至少一个分类，再进行单独友情链接的创建
