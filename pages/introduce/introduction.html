# 介绍

整个系统父子包含关系及相关名词含义解释如下：

# 名词解释：门户

一组包含多个布局组件的展示集合，一组排列好大小及位置的首页。

# 名词解释：布局

可自定义大小及位置的容器，内可以放置一个或多个栏目，栏目的显示区域受布局大小的控制，一个布局为有规则的矩形区域

# 名词解释：栏目

最小单位单元，放置在某个布局内，并可使用系统固定配置好的栏目单独配置相关参数，或使用外链或html代码进行自定义控制。

# 依存(包含)关系

门户 (包含) -> 布局[1个或多个] (包含) -> 栏目[1个或多个]
