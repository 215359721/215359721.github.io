# 门户授权

门户授权功能为将已创建的固定的门户授权给固定的部门使用，以达到不同部门用户使用不同门户的作用

# 授权规则

① 一个门户只可以授权给一个部门（主部门或子部门） ② 已经建立授权对应的门户或部门不能再与其他门户或部门建立对应关系（只能1对1，不能1对多或多对多） ③ 前台用户会根据所属的部门查找最近对应关系的门户进行展示 ④ 默认门户不能建立授权关系 ⑤ 若未在后台建立任何门户授权或用户当前部门未建立授权关系，则当前用户登录前台使用默认门户

# 建立授权

管理后台 -> 门户授权 -> 选择门户 -> 选择部门 -> 点击“建立授权”按钮

# 取消授权

管理后台 -> 门户授权 -> 查看“已关联数据” -> 选择欲取消的关联 -> 点击“取消关联”链接
