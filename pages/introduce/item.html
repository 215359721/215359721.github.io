# 栏目

栏目为组成网站的最小单元单位，也是可定制化程度最高的单位

系统提供了10种内置模块（文字新闻、图片轮播新闻、应用中心、常用服务、待办任务、日程、消息、企业简介、三方外链、自定义代码）可以实现网站大部分展示及应用入口的使用场景

除此之外，外链功能提供三方网站、网页的嵌入；自定义代码功能提供简单逻辑或纯html展示方案的轻度定制，最大化满足门户及网站的日常使用

若10种内置栏目及方案无法满足要求，也可进行二次开发或复杂度高的定制开发，进行栏目的扩充

# 栏目主要属性

# 【栏目名称】

栏目的名称，前台栏目标题

# 【栏目类型】

内置模块选择（文字新闻、图片轮播新闻、应用中心、常用服务、待办任务、日程、消息、企业简介、三方外链、自定义代码）

# 【内容加载方式】

内置模块: 使用固定的内置模块，注意：【四创类】 此处填写四创模块id；【三方外链】此处填写外链url（以http:// 或 https:// 开头）；【其余模块】此处可以填写任意值或直接填写null

代码配置: 通过自定义编辑html代码方式进行模块内容配置，有一定自由度

# 【URL】

仅内置模块需配置，请参照“内容加载方式”-“内置模块”相关配置描述

# 【更多访问地址】

前台模块右上角点击“更多”时，访问的地址，可以是内置模块的路由地址或三方外链地址

特殊模块请遵循如下固定值：【新闻展示】/newsList；【消息】/msgList；【待办任务】/taskList；【企业简介】/av-about

其余模块可自定义该地址

注意

注意：当“是否显示栏目标题”开关为关闭状态时，此地址不生效

# 【最小宽度】

当前栏目支持的最小宽度（1-24）

# 【最大宽度】

当前栏目支持的最大宽度（1-24）

# 【最小高度】

当前栏目支持的最小高度（1-24）

# 【最大高度】

当前栏目支持的最大高度（1-24）

# 【是否显示栏目标题】

是否在前台显示栏目标题，默认关闭，建议文字新闻、应用中心、常用服务、待办任务、日程、消息、企业简介等需要指明标题的模块打开此开关显示标题；建议图片轮播新闻、三方外链、自定义代码等不需要指明标题的模块关闭此开关隐藏标题。

# 【是否是四创栏目】

四创类模块：图片新闻、文字新闻

使用以上模块勾选此选项，其余模块可不勾选

# 添加栏目

栏目管理 -> “新建栏目”按钮 -> 填写栏目内容 -> 点击“确定”完成新建

# 修改栏目

栏目管理 -> 选中需要编辑修改的栏目 -> 点击上方黄色“编辑”图样按钮 -> 修改栏目信息 -> 点击“确定”完成新建

# 查找栏目

栏目管理 -> 通过上方输入框输入要查找的栏目标题（支持模糊查询）-> 回车键查找符合条件数据（输入空位查找全部）

# 删除栏目

栏目管理 -> 选中需要删除的栏目 -> 点击上方红色“删除”图样按钮 -> 弹框确认删除点击“确定”完成栏目的删除

注意

如果该栏目目前已与布局建立关联，则无法删除该栏目，需要先前往门户布局中删除对应的栏目，再在此删除
